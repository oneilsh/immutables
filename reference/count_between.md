# Count elements in a key range

Count elements in a key range

## Usage

``` r
count_between(x, from_key, to_key, include_from = TRUE, include_to = TRUE)
```

## Arguments

- x:

  An \`ordered_sequence\`.

- from_key:

  Lower bound key.

- to_key:

  Upper bound key.

- include_from:

  Include lower bound when \`TRUE\`.

- include_to:

  Include upper bound when \`TRUE\`.

## Value

Integer count of matching elements.
