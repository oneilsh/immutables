# Apply a function over flexseq elements

Apply a function over flexseq elements

## Usage

``` r
# S3 method for class 'flexseq'
fapply(X, FUN, ...)
```

## Arguments

- X:

  A \`flexseq\`.

- FUN:

  Function to apply to each element.

- ...:

  Additional arguments passed to \`FUN\`.

## Value

A new \`flexseq\` with transformed elements.
