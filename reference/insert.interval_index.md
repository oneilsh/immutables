# Insert an element into an interval index

Insert an element into an interval index

## Usage

``` r
# S3 method for class 'interval_index'
insert(x, element, start, end, name = NULL, ...)
```

## Arguments

- x:

  An \`interval_index\`.

- element:

  Element to insert.

- start:

  Scalar start endpoint.

- end:

  Scalar end endpoint.

- name:

  Optional element name.

- ...:

  Unused.

## Value

Updated \`interval_index\`.
