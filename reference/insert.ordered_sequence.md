# Insert an element into an ordered sequence

Insert an element into an ordered sequence

## Usage

``` r
# S3 method for class 'ordered_sequence'
insert(x, element, key, ...)
```

## Arguments

- x:

  An \`ordered_sequence\`.

- element:

  Element to insert.

- key:

  Scalar key for \`element\`.

- ...:

  Unused.

## Value

Updated \`ordered_sequence\`.
